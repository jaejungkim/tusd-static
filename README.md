tusd-static
===========

> Static binaries for the `tusd` command.

## Installation

```sh
$ npm install tusd-static
```

## Usage

```js
const tusd = require('tusd-static')
console.log(tusd.path)
// '/home/werle/repos/little-core-labs/tusd-static/bin/linux/x64/tusd'
```
